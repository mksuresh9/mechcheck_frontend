# MechCheck Mechanic App (Expo)

Starter Expo React Native app for MechCheck mechanic application.

Features scaffolded:
- Booking request list with accept/reject
- Navigation placeholder (Google Maps integration stub)
- Inspection checklist and image upload placeholders
- Report submission and earnings dashboard

Run locally:

```bash
cd mechanic-app
npm install
npm start
```

Notes: Integrations (Maps, notifications, image storage) are mocked and require API keys and server support.
