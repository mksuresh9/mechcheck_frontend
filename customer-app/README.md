# MechCheck Customer App (Expo)

Starter Expo React Native app for MechCheck customer application.

Features included in scaffold:
- Login flow (Auth context)
- Booking flow screens
- Nearby mechanics list
- Live status tracking placeholder
- Inspection report screen
- Profile and ratings

Run locally:

```bash
cd customer-app
npm install
npm start
```

Notes:
- This scaffold uses mocked data for maps/payments. Integrations (Google Maps, Firebase, Razorpay) require additional setup and API keys.
